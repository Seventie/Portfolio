## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/seventie/portfolio.git
cd portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The optimized build will be in the `dist` folder.

## ğŸ¨ Customization

### Colors

The portfolio uses CSS variables for theming. Modify `src/index.css` to customize colors:

```css
:root {
  --background: 0 0% 100%;
  --foreground: 0 0% 3.9%;
  --primary: 0 0% 9%;
  /* ... more variables */
}
```

### Content

Update the content in each component file under `src/components/portfolio/` to personalize:
- Projects in `Projects.tsx`
- Research in `Research.tsx`
- Education in `Education.tsx`
- Tech Stack in `TechStack.tsx`
- Certifications in `Certifications.tsx`
